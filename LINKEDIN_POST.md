🚀 **Just shipped a side project that combines my fintech background with AI — and I'm really proud of how it turned out.**

After years working in payments at Mastercard, I understand one thing deeply: raw transaction data is only valuable when it tells a story.

So I built **TransactIQ** — an AI-powered dashboard that transforms your bank statement into clear, actionable spending insights.

💡 Here's what it does:
→ Upload any CSV bank statement
→ Auto-classifies merchants (Food, Shopping, Travel...)
→ Runs Claude AI to generate natural-language insights like: *"Your weekend spending is 2.4x your weekday average — mostly food delivery"*
→ Visual breakdowns: weekly trends, category charts, transaction search

🛠️ **Tech stack:**
• React + Recharts (frontend)
• Spring Boot + PostgreSQL (backend)
• Anthropic Claude API (AI insights)
• Docker Compose (one-command deploy)

This project helped me bridge two worlds I care about: **enterprise-grade fintech architecture** and **consumer-friendly AI products**.
